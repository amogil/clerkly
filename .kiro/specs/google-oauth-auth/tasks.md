#### 1.1 Реализовать логику обработки истекших refresh tokens
- [ ] Добавить функцию `handleExpiredRefreshToken` в main.ts:
  ```typescript

    sendAuthResultToRenderer({ 
      success: false, 
      error: "Сессия истекла. Войдите заново." 
    })
  }
  ```
