# Clerkly - Интерфейс приложения

## Обзор

Интерфейс Clerkly построен на React с использованием современных UI компонентов и Tailwind CSS. Приложение имеет адаптивный дизайн с боковой панелью AI-ассистента и основной рабочей областью.

## Структура интерфейса

### Главные компоненты

1. **Top Navigation** - Верхняя навигационная панель
   - Переключение между разделами приложения
   - Индикатор текущего раздела
   - Компактный дизайн

2. **AI Agent Panel** - Боковая панель AI-ассистента (правая сторона)
   - Занимает 33.33% ширины экрана
   - Чат с AI ассистентом
   - История задач и команд
   - Статусы выполнения задач

3. **Main Content Area** - Основная рабочая область
   - Занимает 66.67% ширины экрана
   - Динамически меняется в зависимости от выбранного раздела

## Разделы приложения

### Dashboard (Главная)
- Обзор предстоящих встреч
- Список активных задач
- Быстрый доступ к важным функциям
- Статистика и метрики

### Calendar (Календарь)
- Просмотр встреч и событий
- Навигация по датам
- Детальная информация о встречах
- Интеграция с AI для анализа встреч

### Tasks (Задачи)
- Управление проектами и задачами
- Создание новых задач
- Отслеживание прогресса
- Фильтрация и сортировка

### Contacts (Контакты)
- Список контактов
- Поиск по контактам
- Добавление новых контактов
- Информация о контактах

### Settings (Настройки)
- Настройки приложения
- Управление аккаунтом
- Интеграции
- Выход из системы

## AI Agent Panel

### Возможности

1. **Чат с AI**
   - Отправка команд и запросов
   - Получение ответов и рекомендаций
   - История диалога

2. **Задачи AI**
   - Просмотр активных задач AI
   - Статусы выполнения:
     - `new` - Новая задача
     - `in-progress` - В процессе выполнения
     - `awaiting-user` - Ожидает действий пользователя
     - `completed` - Завершена
     - `error` - Ошибка выполнения

3. **Команды**
   - Навигация: `navigate to [screen]`
   - Создание: `create [entity]`
   - Просмотр: `show [entity]`

### Примеры команд

```
navigate to tasks
create new project
show calendar
create contact
```

Приложение использует библиотеку Radix UI для базовых компонентов:

- **Button** - Кнопки различных стилей
- **Input** - Поля ввода
- **Select** - Выпадающие списки
- **Dialog** - Модальные окна
- **Tabs** - Вкладки
- **Card** - Карточки контента
- **Avatar** - Аватары пользователей
- **Badge** - Бейджи и метки
- **Tooltip** - Подсказки
- И многие другие...

## Стили и темы

### Цветовая схема

Приложение поддерживает светлую и темную темы с использованием CSS переменных:

- `--background` - Фон приложения
- `--foreground` - Основной текст
- `--primary` - Основной цвет (#5b6cf2)
- `--secondary` - Вторичный цвет
- `--accent` - Акцентный цвет
- `--muted` - Приглушенный цвет
- `--destructive` - Цвет для деструктивных действий

### Шрифты

Используется шрифт **Inter** от Google Fonts с различными начертаниями (300-700).

### Адаптивность

Интерфейс адаптируется под различные размеры окна:
- Минимальная ширина: 800px
- Минимальная высота: 600px
- AI Panel всегда занимает 33.33% ширины
- Main Content адаптируется под оставшееся пространство

## Интеграция с Electron

### IPC коммуникация

Интерфейс взаимодействует с main процессом через IPC:

```typescript
// Доступно через window.api (определено в preload)
window.api.saveData(key, value)
window.api.loadData(key)
window.api.deleteData(key)
```

### Безопасность

- Context Isolation включен
- Node Integration отключен
- Sandbox режим активен
- Все IPC вызовы проходят через безопасный preload script

## Разработка

### Добавление нового компонента

1. Создайте файл в `src/renderer/components/`
2. Используйте TypeScript и React
3. Следуйте существующим паттернам
4. Добавьте типы для props
5. Используйте Tailwind CSS для стилей

### Пример компонента

```typescript
// Requirements: clerkly.1
interface MyComponentProps {
  title: string;
  onAction: () => void;
}

// Requirements: clerkly.1
export function MyComponent({ title, onAction }: MyComponentProps) {
  return (
    <div className="p-4 bg-card rounded-lg">
      <h2 className="text-xl font-semibold">{title}</h2>
      <button onClick={onAction} className="mt-4 px-4 py-2 bg-primary text-white rounded">
        Action
      </button>
    </div>
  );
}
```

### Использование UI компонентов

```typescript
import { Button } from './components/ui/button';
import { Input } from './components/ui/input';
import { Card } from './components/ui/card';

// Requirements: clerkly.1
export function MyForm() {
  return (
    <Card>
      <Input placeholder="Enter text" />
      <Button>Submit</Button>
    </Card>
  );
}
```
