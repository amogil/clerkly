# Clerkly

Clerkly is a desktop AI agent for everyday users.  
Install the app, add your OpenAI API key, and start in minutes.

## What Is Clerkly

Clerkly gives you one focused workspace to:

- ask for clear summaries
- turn scattered updates into clear next actions
- keep decisions and context in one place

Built local-first: your working data stays on your machine.

## Why People Need It

Most people lose hours every week to:

- constant context switching
- repeated status requests
- unclear ownership and next steps after meetings

Clerkly turns that into:

- clear summaries
- explicit priorities
- faster alignment with less back-and-forth

## Value You Feel Quickly

- Less context switching
- Faster daily planning and prioritization
- Better continuity between meetings
- More time for real work

## Security First

Security is a core product value in Clerkly:

- All work happens locally on your device
- Agent code runs in an isolated browser sandbox
- The sandbox is isolated from direct network access and from your local file system
- The agent has no direct access to your passwords, API keys, secrets, or other sensitive local data, so sandboxed code cannot leak them to the network
- The agent cannot perform sensitive or potentially harmful actions without your explicit approval

You stay in control of what the agent can and cannot do.

## System Requirements

- macOS 10.13 or newer
- 500 MB free disk space

## Install

1. Download the latest Clerkly release from GitHub Releases.
2. Open the `.dmg` file.
3. Drag Clerkly to `Applications`.
4. Launch Clerkly.
5. Add your OpenAI API key in settings.
6. Start working.

No server setup, Docker, or complex installation is required.
